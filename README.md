# sos-202

  **Install all components trough Bower:**
    ```
    bower install
    ```

 To launch local server use node.js and node manager package (nmp).

 **Install http-server globally:**
   ```
   npm install http-server -g
   ```

 **And launch it:**
   ```
   http-server
   ```

 **Then, project will be available on:**
   ```
   http://127.0.0.1:8080
   ```
